# Plans LQ

Sell subscriptions on your Wix site through **your own Stripe account**, with
member roles and contact labels granted automatically via a single Stripe
webhook.

## Get started

1. **[Installation](./installation.md)** — install the app + package and wire the backend.
2. **[Connect Stripe](./connect-stripe.md)** — connect your Stripe account and finish setup.
3. **[Creating & displaying plans](./plans-setup.md)** — create plans and add them to your page.
4. **[Settings](./settings.md)** — fine-tune checkout & Billing Portal behavior.

## Add-on features

- **[Usage limits](./usage-limits.md)** — cap how many items a member can create, per plan.

## More

- **[Testing](./testing.md)** — run a full subscribe-to-role flow with a Stripe test account.
- **[For developers](./developers.md)** — extension points such as a custom webhook logger.
